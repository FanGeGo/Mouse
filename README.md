Mouse(打地鼠)
===========

![](https://raw.githubusercontent.com/muyangren907/Mouse/master/image/screenshots.jpg)

## 说明

- 纯Java编写的打地鼠小游戏，仅供参考

## 使用

### 编译运行

- 下载源代码

- 编译

```
javac Mouse.java -d ./ -encoding utf-8
```

-	运行

```
java Mouse.Mouse
```
